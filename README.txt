# Coordinate Extractor

## Installation
No installation required. Just extract all files keeping the folder structure.

## Starting the Application
1. Double-click on "Start_CoordinateExtractor.bat" to start the program

## Usage
1. Select two images containing coordinate information
2. Click "Compute Distance" to calculate the distance between the coordinates

## Important Notes
- Do not move or delete the Tesseract-OCR folder
- Keep the original folder structure
- The application needs to access Tesseract-OCR to function properly
